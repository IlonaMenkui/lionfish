* [Telegraf](http://telegraf.js.org/)

## Usage

### Local

The following environment variables should be exported before running:
* `LF_APP_API_URL`
* `LF_APP_TELEGRAM_BOT_TOKEN`
* `LF_LB_URL`

Start application:
```
npm start
```

Start application in debug mode:
```
npm debug
```
